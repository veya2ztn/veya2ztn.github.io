Wang-Landau Sampling

Posted on 2017-01-05

Wikipedia: Wang and Landau algorithm

Simple History

In the history of Wang-Landau algorithm, it is first used for calculate the free energy or the partition function for a statistic system. Because for an equilibrium state, all the property of system can derivate from thermodynamic quantity like free energy.

Then, due to its property that can fast visit all the state, this method is used for jumping out from local min trap and to find global min.

Main Idea

If a system can be regard as a physics statistic canonical system and every configuration map to a certain cost (energy). Then we can use the state density to suggest the probability of energy. More commonly, a canonical system is at energy $E$ and $E+dE$ with probability
$$
p(E)dE=g(E)\exp(-\beta E)dE
$$
This is all the thing where we start!

Ordinary Monte-Carlo Sampling

Before we talk about the nature of Wang-Landau Sampling, it is necessary to discuss the normal processing our physicists using to calculate the mean value (or observation value) —— Monte-Carlo Sampling

For this purpose, the Monte-Carlo is only use for sampling a probability distribution, like:

if we want to calculate the average of physics quantity $A$
$$
\left=\int_{all} A(\vec{r})\frac{e^{-\beta E(\vec{r})}}{Z}d\vec{r}
$$
where $Z$ is the partition function
$$
Z= \int_{all} e^{-\beta E_{\vec{r}}}d\vec{r}
$$
we can regard $\frac{e^{-\beta E(\vec{r})}}{Z}$ as a term for probability $\rho(\vec{r})$ so that
$$
\left=\int_{all} A(\vec{r})\rho(\vec{r})d\vec{r}
$$
So that if the variable $A$ happened as $\rho$, we can get $\left$ directly.

computing drawback

It is obvious that we can compute one $\left$ every time we set a $\beta=\frac{1}{k_bT}$.

WL Sampling

We can use another formula to calculate average
$$
\left=\int_{all} A(E)g(E)e^{-\beta E}dE
$$
notice that $A(\vec{r})$ now become function of energy $E$.

Since the $g(E)$ is independent from temperature $T$. Why not set $\beta=0$？

Now, we arrive Wang and Landau thought!

if $\beta=0$, the probability for canonical system at energy $E$ and $E+dE$ become
$$
p(E)dE=g(E)dE
$$
which is mean, now the energy distribution follow $g(E)$.

So, we start to talk about main idea of feasibility!

  1. We know that if the system is under canonical distribution (detailed balance), the energy distribution follow $\rho(E)=g(E)$. So, if we using $\frac{1}{g(E)}$ as a sampling possibility for each $E$ we will get a flat histogram!

  2. But, of course, the system cannot be the correct energy distribution, and the $g(E)$ which is regard as the possibility of distribution $E$ is depend on distribution $E$

    $$
    g(E) \leftrightarrow Distribution
    $$

  3. If we apply a function $\mathfrak{F}$, which is $g_{n+1}=\mathfrak{F}(g_n)$ and it can always converge to $g_N=g_{aim}$

That is make our sampling success.

Follow the processing provided by Wang-Landau Algorithm
$$
p(E_1\rightarrow E_2)=\min [\frac{g(E_1)}{g(E_2)},1]
$$
And modify the $g(E)$ using — if $E_i$ is chosen to jump
$$
g(E_i)=factor \times g(E_i)
$$
And convergence processing — after certain step
$$
f\leftarrow f/2
$$
we hope we can believe that our method is universal.

However, no one proved its analysis convergence and it obviously exists some problems.

First, the jumping rule
$$
p(E_1\rightarrow E_2)=\min [\frac{g(E_1)}{g(E_2)},1]
$$
must be correct followed
$$
p(E_1\rightarrow E_2)=\min [\frac{\rho(E_2\rightarrow E_1)/g(E_2)}{\rho(E_1\rightarrow E_2)/g(E_1)},1]
$$
The $\rho(E_1\rightarrow E_2)$ and $\rho(E_2\rightarrow E_1)$ is intrinsic possibility for one particle jumps from this energy to another, we always set them are equal $\rho(E_1\rightarrow E_2)=\rho(E_2\rightarrow E_1)$ but no one can guarantee that.

Second, It was later shown that updating the f by constantly dividing by two can lead to saturation errors.A small modification to the Wang and Landau method to avoid this problem is to use the f factor proportional to $1/t$, where $t$ is proportional to the number of steps of the simulation.

Remarkable Point

The remarkable trick is using $e^{S(E)}$ to replace $g(E)$ and then we compute
$$
\log p(E_1\rightarrow E_2)=\min [S(E_1)-S(E_2),1]
$$

$$
S(E_i)=S(E_i)+\bar{f}
$$

This trick can simplify computation and what more:

The exponent form means that the jumping possibility ( or the jumping wall ) are exponent increase! Which can be regard as a reason that why WL sampling can quickly visiting all the available energy spectrum.

Why it can used for not statistic system?

Up to know, what we only talk about is that how the WL sampling useful for statistic physics. But, actually, many people try to apply it to find the global min problem, and maybe other problem.

Why? That is because we have the brief — believe its continues.

Remember the natural power of this sampling:

Quickly Visiting ALL the available energy spectrum.

Let me show you an instinctive example.

First, we need to find the min of a very smooth function.

Using the simple local min method ( if it is a multi-variable function, using, e.g, GFBS ), we can easily get the min position

However, in practice, ( for most complex multi-variable problem), the function is not smooth and such bumpiness. (for multi-variable, image is not 1D but the bumpiness is can be imaged.)

How to get the global min?

If we can search the red point and then using determined optimization, we can actually arrive to global min.

However, why it will happen? For what, we can say that “I must touch the critical point once or more”?

Wang-Landau Sampling can Quickly Visiting ALL the available energy spectrum!

If we sampling the energy between a suitable interval, e.g in the picture from $E=3​$ to $E=5​$, WL guarantee that after enough searching, we can visit all the state in the set $3<E<5​$. Since the red points are also in the $3<E<5​$ we can safely say that we find the global min.

Here is the histogram for state density.

That is why we believe the WL sampling can help us to solve global optimization problem.
