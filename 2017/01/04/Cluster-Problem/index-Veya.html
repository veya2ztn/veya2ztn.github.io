Cluster Problem

Posted on 2017-01-04

The Cluster Problem is a basic physics optimization problem, which only involved:

  • Numbers of particles
  • Interaction according coordinate, spin or other intrinsic properties

Ana the directly question for this problem is to find out the min energy structure (site distribution, spin mode, etc.)

Here is a typical Leonard-Jones Cluster problem
$$
find {(X1,X2,\dots,X3)|\min \sum_{ij}(\frac{1}{r_{ij}^{12}}-\frac{2}{r_{ij}^6}),r_{ij}=|X_i-X_j|}
$$

We can define any Interaction function for this system, however, only small subset of function has a good physics mapping like Leonard-Jones Potential.

All the question can be classified as a large multi-variable function optimization problem. If we do not consider spin or other separate variable. This a typical continues optimization problem. What is difficult for this problem is not to find a local min, otherwise, how to find the global min for a huge particle system.

Unluckily, there seldom are problems can be analyzed. The majority of solution is numerical method. There are many means to search a global min.

  • molecular dynamics simulation
  • specific random sampling

I deliberately using the “specific” to named a series of means like Monte-Carlo with different sampling rule and other heuristic method like Artificial Bee Colony. They are all sampling method which rely on the compute’s fast repeating property.

However, please note that:

Now, Lennard-Jones clusters have become a much-studied test system for global optimization methods designed for configurational problems.

There are many researches for this problem:

  1. Structural optimization of Lennard-Jones clusters by a genetic algorithm
  2. Minimization of the potential energy surface of Leonard–Jones clusters by quantum optimization
  3. GLOBAL OPTIMIZATION OF LENNARD-JONES ATOMIC CLUSTERS
  4. Global Optimization by Basin-Hopping and the Lowest Energy Structures

And it is wonderful that Cambridge Cluster Database has already list all the possible results.
